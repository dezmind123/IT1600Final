## What Is a File Type?

A **file type** describes the structure and purpose of a file, and how to interpret the data the computer is being given. 



A few examples of file types and a high-level overview of what they are used for:  
- `.txt` → plain text  
- `.jpeg` → compressed image  
- `.mp3` → audio  
- `.exe` → executable program
- `.json` → storing data
- `.xml` → text for markup lanaguge
- `.png` → photos
- `.jpeg` → images
- `.mpeg-4` → audio

---

## What Is a File Extension?

A **file extension** is the characters after the dot after the filename. It determines the file type and how it is saved on the computer, and how it can be opened.
