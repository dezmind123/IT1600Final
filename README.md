# Choosing the file type for you

**Author:** Desmond Jones

**Summary:**  
This short tutorial will give you a basic understanding of various file types and how to choose the best one for you. We will explore various types of files and their best use cases. At the end of the tutorial, the reader should be able to identify different types of files and find the one best suited for them. 

**Target audience:**  
While this tutorial does not have a specific target demographic, it would be more useful for anyone who wants ot learn more about file types and is searching to see what file type would best suit their project. 

**Estimated completion time:** ~6 minutes.
