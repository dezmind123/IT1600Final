## What Is a File Type?

A **file type** describes the structure and purpose of a file, and how to interpret the data the computer is being given. 

- `.JSON`
  <p>
JSON is a lightweight file format to store and transport data. It stands out for having   allowed for simple communication and simplicity of sending, receiving, and storing data. </p>

- `.XML`
XML is a text-based document format that uses Extensible Markup Language. It uses tags to predetermine controls for content and allows users to define/add their own. This allows for highly flexible data types 

- `.MP3`
MP3 is a digital audio file typically containing music that is compressed, allowing for easy storage by compressing the needed data without degrading the quality

- `.PNG`
PNG is designed to create a robust raster image file that typically contains photographs. Commonly designed to work with viewing applications on the internet. 

- `.JPEG`
While mostly complying with images and the most common file format among them. It is typically used on digital cameras and phones.

- `.MPEG-4`
MPEG-4 files are digital multimedia formats that compress audio and video efficiently while keeping high quality for streaming and storage, and they also support advanced features like interactivity and 3D content.
